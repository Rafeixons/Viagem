📌 Descrição
Programa em Java com JOptionPane que simula o planejamento de uma viagem. O usuário informa nome, data, dias e valor por dia; o sistema valida os dados, realiza cálculos e mostra um resumo da viagem.

🎯 Objetivos
Entrada e saída com JOptionPane

Conversão de String para números

Uso de LocalDate

Validação de dados

Cálculos simples

⚙️ Regras
Menu inicial: Planejar viagem ou Sair

Solicitar: nome, data (dd/MM/yyyy), dias (int), valor/dia (double)

Validar: campos vazios, números, datas, valores negativos

Processar:

Total = dias × valor/dia

Situação: passada, hoje ou futura (com dias restantes)

🖥️ Exemplo
Entrada:

Nome: Sebastiana

Data: 20/12/2026

Dias: 5

Valor/dia: 150.50.
